# Уроки Python

Краткий вводный курс, охватывающий основы Python через практические примеры.

## Уроки

### 1. **Типы данных** (`data-types.py`)
Введение в основные типы данных Python, включая:
- Логические значения (`True`, `False`)
- Числовые типы: целые числа, двоичные, восьмеричные и шестнадцатеричные числа
- Комплексные числа
- Строковые литералы и многострочные строки
- Форматирование строк и интерполяция (f-строки)
- Сырые строки и обработка экранирующих символов

### 2. **Динамические типы** (`dyn-types.py`)
Исследование системы динамической типизации Python:
- Как переменные могут содержать разные типы в разные моменты
- Проверка типа с функцией `type()`
- Преобразование и приведение типов (int, str и т.д.)
- Работа с пользовательским вводом и преобразованием типов
